# Archimède — One Command Demo

This repository contains a minimal "one command" demo that runs experiments using Grid2Op and simple models.

Structure

- archimede_demo/: Python package with environment adapters, models, training and evaluation scripts.
- run.sh: convenience script to run the demo (quick or full).
- requirements.txt: pinned dependency list.

Quick start (after installing requirements):

./run.sh --quick

See archimede_demo/README for developer notes.
